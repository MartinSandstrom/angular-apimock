Simple demo

Let's pretend. You love Magic the Gathering cards and you're going to create an online database where people can search for different cards. You know you want to load it with AJAX, but you don't know what you should include yet so you can't write the API just yet. But you still want to create the front-end as if there was one!

It would look something like the below. Try typing something into the search field.

It failed because there's no real API. Now add ?apimock=true to the URL in your browsers addressbar. Right there at the top of the page, so it says http://johansson.jp/angular-apimock/#/demo-simple?apimock=true. Then click the button again.

If you already changed your URL and see this, you might have to refresh the page as it seems Angular routing doesn't really refresh. Looking into fixing this.
